# Performance evidence registry

`baseline.json` is the checked-in, machine-readable performance evidence for every production AdapterCapability. It was generated by `tools/benchmark_capsules.py` on the environment recorded inside the report.

Consumers must match records by `capability_id`; array position is not an identity. A manifest evidence URI has the form:

```text
registry/performance/baseline.json#capability:<capsule>/<strategy>/<backend>
```

Use `cost_model` for graph planning. `score.overall_0_to_100` is a calibrated summary for same-profile ranking and must not override semantic, loss, security or resource constraints. See `docs/13-performance-evidence.md`.
