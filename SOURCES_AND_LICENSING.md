# Sources and licensing boundary

This foundation artifact contains original EverythingX ontology text, schemas, source manifests, and normalized factual indexes. It does not redistribute the downloaded upstream raw archives.

Upstream names, identifiers, media types, filename patterns, and terse structural facts remain attributable to their listed sources. Each upstream source has its own terms, policies, and project license. Before publishing a derived database or bundling raw snapshots in a product, review the current terms at the documentation URL recorded in `sources/sources.json`.

The catalog builder deliberately does not copy long descriptive prose from Library of Congress FDD, IANA notes, or project comments. Source records retain a locator so a human or later synchronization job can consult the authoritative record.

No claim is made that an upstream registry's inclusion endorses a format, that omission means a format does not exist, or that two records sharing a MIME type or extension are identical.
